## Introduction

This module enhances the Order Details section in the admin area by providing additional features and controls:

* **Error Display:** Shows errors directly below the order details for quick reference
* **Processing Section:** Adds a dedicated section beneath the order details to streamline order processing
* **Metadata Exclusion:** Prevents specific metadata from appearing on individual order items, reducing clutter
* **Metadata Search Support:** Enables search functionality using the order ID within metadata
* **Default Item Naming:** Automatically assigns default names to order items that lack a name


## Installation

* Run composer `require woosa/order-details:<version>` in the plugin's root directory. Alternatively, you can add `"woosa/order-details": "<version>"` directly in `composer.json` and then run `npm start`
